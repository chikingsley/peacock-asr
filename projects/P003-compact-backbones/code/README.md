# P003 Code

This directory is for compact-backbone project-local code, wrappers, and helper
scripts.

Rules:

- Keep `P003`-specific training, preprocessing, launchers, and benchmark
  wrappers here.
- `training/` is the canonical home of the compact-backbone phoneme-head
  pipeline.
- Keep code in `P003` unless there is a deliberate later extraction into a
  separate shared package.
